  • SNEX

    SNEX is a project started with Ethan Dirks which aims to provide classic style game pads that render on smartphones and send JSON encoded events to web applications. You can implement them in your app.

  • Megaman 2 JS

    A remake of Megaman 2 I started in order to learn about game programming. You can try it. Play with W-A-S-D, O-P, Q-E or remap using the on-screen remapper.

  • Snakesilk

    A game engine for Web environment running Javascript. It's open sourced and installable via NPM. This engine was made during Megaman 2 JS development and decoupled in 2017.

  • Meth Meth Method

    A YouTube show where I teach Javascript inspired by Fun Fun Function. All projects available on GitHub.

  • Grafit

    A graphing tool for visualizing Javascript function output.

  • Keycode

    A tiny tool that shows the keycode of pressed keyboard key.
